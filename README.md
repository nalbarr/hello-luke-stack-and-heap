# hello-luke-stack-and-heap
hello-luke-stack-and-heap

### Setup
- Need Linux, OSX with Python, Java, C++ environments available at command line.
- Refer to Makefiles to see specifics.

### How to run

#### Python
`cd python; make run`

#### Java
`cd java; make run`

#### C++
`cd cpp; make run`
